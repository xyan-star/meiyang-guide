# 美羊AI学习力系统新手向导

基于 Vue 3 + Element Plus 构建的新手引导页面。

## 线上地址

🔗 **https://xyan-star.github.io/meiyang-guide/**

## 功能

- 注册账号
- 系统登录
- 快速了解（教师/学生手册、汇享资源）
- 功能导览
- 在线测试（4套知识问卷，答题即时出分，推送企业微信）

## 部署

部署于 GitHub Pages，push 到 `master` 分支自动生效。

## 技术栈

- Vue 3 (CDN)
- Element Plus (CDN)
- Font Awesome (CDN)
